## Como rodar o projeto

### Instalar as dependencias

```bash
$ npm install
```

### Comando para dar start

```bash
# 
$ npm run start
```

### Doker compose

```bash
# 
$ docker compose -f docker-compose.yml up --build -d
```

### Doker

```bash
# 
$ docker build -t container . && docker run -it container
```
